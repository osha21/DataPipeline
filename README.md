# DataPipeline
The function of this repository is to analyze data acquired for the first semester of the master's in Medialogy at AAU

The analysis consists of categorizing the participants in four profiles across their level of agency and engagement. 

Principle Component Analysis was used to analyze the telemetric data, along with Fisher's Exact Test to find association between the this data and the profiles.

A correlation matrix was computed to verify the loadings of the PCA and find the variance within the data for the PCA
